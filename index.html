ClickGrapher

A Click (Python CLI) tool for drawing graphs from files.

View on GitHub Download .zip Download .tar.gz

Present Graphs in Terminal.

Draws a graph from a given points in a file. Automatically scales graph's viewpoint and optionally connects the points.

DOCS

Usage:

grapher [OPTIONS] FILE

Options:

height

Define the height of a graph. --height INTEGER or -h INTEGER

-h, --height INTEGER Number of characters used to display height of a graph. -w, --width INTEGER Number of characters used to display width of a graph. -s, --step INTEGER How wide should each space be.
