ClickGrapher

A Click (Python CLI) tool for drawing graphs from files.

Download as .zip Download as .tar.gz View on GitHub

Present Graphs in Terminal.

Draws a graph from a given points in a file. Automatically scales graph's viewpoint and optionally connects the points.

INSTALL

pip install .

DOCS

Usage:

grapher [OPTIONS] FILE or python grapher.py [OPTIONS] FILE

Options:

height

Define the height of a graph.

--height INTEGER or -h INTEGER

width

Define the maximum width of a graph.

--width INTEGER or -w INTEGER

step

Define the width of each unit.

--step INTEGER or -s INTEGER
